1️⃣ What is the difference between var, let, and const?
Ans: 
i) var → function scoped, can be re-declared and re-assigned, hoisted.
ii) let → block scoped, can be re-assigned but not re-declared, hoisted (not initialized).
iii) const → block scoped, cannot be re-assigned or re-declared, hoisted (not initialized).

2️⃣ What is the spread operator (...)?
Ans:
i) Expands elements of arrays or objects.
ii) Used for copying, merging, or passing multiple values.

3️⃣ What is the difference between map(), filter(), and forEach()?
Ans: 
i) map() → returns a new array after transforming each element.
ii) filter() → returns a new array with elements that pass a condition.
iii) forEach() → executes a function on each element, returns nothing.

4️⃣ What is an arrow function?
Ans:
i) Shorter syntax for writing functions.
ii) Example: const add = (a, b) => a + b;
iii) Does not have its own this.

5️⃣ What are template literals?
Ans:
i) Strings enclosed in backticks `.
ii) Can embed variables with ${variable}.
iii) Supports multi-line strings.
